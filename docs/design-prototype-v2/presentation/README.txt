GovTech DE — Presentation Bundle
=================================

Two self-contained HTML files for showing the project to another model
or human reviewer.

  pitch.html    — single-page editorial brief with embedded UI mockups,
                  personas, design foundations and 14 Hard-Lines.
                  Self-contained except for Google Fonts (Source Sans 3).

  ui-kit.html   — fully working click-thru prototype of the product.
                  Persona picker → Dashboard → Posteingang → Letter
                  detail → Stammdaten → Umzug autopilot.
                  Self-contained except for React / ReactDOM / Babel
                  (loaded from unpkg.com — needed to run JSX in browser).

Both files run offline once their respective CDNs are cached. Drag-and-drop
into a browser to open, or upload directly to a reviewer.

Source repo: https://github.com/klarbau/govtech-de
All data shown is synthetic. No real Behörden are integrated.
